# คู่มือการติดตั้งและใช้งาน

## ขั้นตอนการติดตั้ง

### 1. ติดตั้ง Dependencies

```bash
npm install
```

### 2. เริ่มต้นใช้งาน

```bash
npm run dev
```

คำสั่งนี้จะเริ่มทั้ง:
- Next.js frontend บน http://localhost:3000
- Express backend บน http://localhost:3001

### 3. เข้าสู่ระบบ

เปิดเบราว์เซอร์และไปที่ http://localhost:3000

ใช้ข้อมูลเริ่มต้น:
- **Username**: admin
- **Password**: admin123

## คุณสมบัติหลัก

### 1. โหมดตาราง (Table View)
- ดูข้อมูลรถทั้งหมดในรูปแบบตาราง
- เพิ่ม แก้ไข ลบข้อมูลรถ
- นำเข้าข้อมูลจากไฟล์ Excel
- ส่งออกข้อมูลเป็นไฟล์ Excel

### 2. โหมดแผนที่ (Map View)
- แสดงรถบน Canvas แบบ Interactive
- ลากรถเพื่อวางตำแหน่ง
- หมุนรถด้วยการคลิกจุดสีเขียว
- ค้นหารถและโฟกัสไปที่ตำแหน่ง
- แสดงมุมการหมุนของรถ

### 3. การทำงานแบบ Real-time
- อัปเดตข้อมูลทันทีเมื่อมีการเปลี่ยนแปลง
- ผู้ใช้หลายคนสามารถทำงานพร้อมกันได้
- ใช้ WebSocket สำหรับการซิงค์ข้อมูล

### 4. ระบบลานจอดหลายลาน
- แยกข้อมูลตามลานจอด
- เลือกดูเฉพาะลานที่ต้องการ
- ค้นหาข้อมูลข้ามทุกลาน

## การนำเข้าข้อมูลจาก Excel

รูปแบบคอลัมน์ที่รองรับ:

1. Id
2. Start time
3. ลานจอด
4. ชื่อลานจอด
5. โซนจอดรถ เช่น A1
6. เลขทะเบียนเช่นกก-111
7. จังหวัด
8. ยี่ห้อ
9. รุ่นรถ
10. สีรถ เช่น ขาว
11. มีกุญแจหรือไม่ เช่น มี, ไม่มี, เปิดประตูไม่ได้ หรืออื่นๆ (อธิบาย)
12. หมายเหตุ อื่นๆ เช่น ยกไปซ่อมเอง, ยกไปอู่ชื่อ..... , ย้ายจาก A1 ไป B1

## การใช้งานบนมือถือ

แอปพลิเคชันรองรับการใช้งานบนมือถือ:
- เมนูแบบ Hamburger สำหรับหน้าจอเล็ก
- ปุ่มควบคุมที่เหมาะสมกับการสัมผัส
- Layout ปรับตามขนาดหน้าจอ

## คำแนะนำการใช้งาน Map View

1. **ย้ายรถ**: คลิกและลากรถไปยังตำแหน่งที่ต้องการ
2. **หมุนรถ**: คลิกจุดสีเขียวด้านบนรถและลากเพื่อหมุน
3. **ซูม**: ใช้ล้อเมาส์หรือปุ่ม +/- เพื่อซูมเข้า-ออก
4. **เลื่อนแผนที่**: คลิกและลากพื้นที่ว่าง
5. **ค้นหารถ**: พิมพ์ทะเบียนหรือข้อมูลอื่นในช่องค้นหา

## Troubleshooting

### ไม่สามารถเชื่อมต่อ Backend
- ตรวจสอบว่า Express server ทำงานบน port 3001
- ตรวจสอบ console สำหรับข้อผิดพลาด

### Real-time ไม่ทำงาน
- ตรวจสอบการเชื่อมต่อ WebSocket
- รีเฟรชหน้าเว็บ

### นำเข้า Excel ไม่สำเร็จ
- ตรวจสอบรูปแบบคอลัมน์ให้ตรงกับที่กำหนด
- ตรวจสอบว่าไฟล์เป็น .xlsx หรือ .xls

## การ Deploy Production

1. Build แอปพลิเคชัน:
```bash
npm run build
```

2. เริ่มใช้งาน:
```bash
npm start
```

3. เปลี่ยน JWT_SECRET ใน server/index.js
4. ตั้งค่า environment variables ที่เหมาะสม
